# Contribuer a TutoDeCode

Merci de votre interet. Ce projet privilegie la qualite, la sobriete et la souverainete numerique.

## Avant de proposer une contribution
- Lisez le CLA: voir [CLA.md](CLA.md)
- Respectez la garantie zero telemetrie (voir [LEGAL.md](LEGAL.md))
- N'utilisez pas la marque ou le logo sans autorisation (voir [LEGAL.md](LEGAL.md))

## Flux de travail
1) Creez une issue (bug ou feature)
2) Proposez une PR concise, avec contexte et tests si possible
3) Attendez la validation d'un mainteneur

## Standards
- Aucun ajout de telemetrie, trackers, ou SDK publicitaires
- Preferer des dependances locales et auditees
- Documentation claire et concise

## Tests
- `npm run build` pour valider la build web
- `npm run tauri:build` pour valider la build desktop

Merci pour vos contributions.
