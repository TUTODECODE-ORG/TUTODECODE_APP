# Politique de securite

Merci de signaler toute faille de securite de maniere responsable.

## Comment signaler une vulnerabilite
- Envoyer un e-mail a: security@tutodecode.org
- Decrire clairement le probleme, les etapes pour reproduire, et l'impact.
- Eviter la publication publique avant resolution.

## Delais
- Accuse de reception sous 72 heures.
- Evaluation initiale sous 7 jours.
- Correctif ou plan de mitigation communique des que possible.

## Portee
Cette politique couvre le code officiel du projet TUTODECODE et les distributions publiees par l'Association TUTODECODE.

Merci pour votre aide.
